# 17 - Mockups y diseño conceptual de pantallas

## 1. Objetivo

Este documento define los primeros diseños conceptuales (mockups) de AGAFONA App.

No representan el diseño final, sino una guía visual para comprender la distribución de elementos y la experiencia de usuario prevista.

Los diseños estarán adaptados principalmente a dispositivos móviles.

---

## 2. Filosofía general

La aplicación deberá transmitir:

- Naturaleza.
- Fotografía.
- Sencillez.
- Cercanía.
- Profesionalidad.

La experiencia deberá ser intuitiva para cualquier socio independientemente de sus conocimientos tecnológicos.

---

## 18. Evolución futura

En fases posteriores estos mockups podrán transformarse en:

- Wireframes detallados.
- Diseño UI completo.
- Prototipos interactivos.
- Diseño definitivo para programación.

---

## 19. Beneficios

Los mockups permiten:

- Visualizar la aplicación.
- Detectar problemas de navegación.
- Validar ideas antes de programar.
- Ahorrar tiempo de desarrollo.

---

## 20. Conclusión

Estos diseños conceptuales representan la primera aproximación visual de AGAFONA App.

Servirán como referencia para el diseño definitivo de la interfaz y para la posterior fase de desarrollo de la aplicación.
